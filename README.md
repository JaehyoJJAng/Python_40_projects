# SSS급 일자러를 위한 파이썬 40개 프로젝트
> 파이썬 실력 향상을 위한 40개의 프로젝트 수행

- 기간 : 2022년 12월 21일(수) 부터
- 참고 교재 : SSS급 일자러를 위한 파이썬과 40개 프로젝트
- 언어 : Python
***
### 진행 방식
- 1일 1회의 프로젝트 수행
***

### GIT
- 수행했던 프로젝트 Commit

    - 프로젝트명의 폴더명 / 프로젝트에 필요한 파일 / 불필요한 파일은 .gitignore에 기입
</br>

- 커밋 규칙
    - 파일 명
        -     프로젝트명
        </br>
    
    - 파일 추가
        -     [ADD] 프로젝트명
    </br>
  
    - 파일 수정
    
        -     [UPD] 프로젝트명
