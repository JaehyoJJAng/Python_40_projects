### GIT
- 수행했던 프로젝트 Commit

    - 프로젝트명의 폴더명 / 프로젝트에 필요한 파일 / 불필요한 파일은 .gitignore에 기입
</br>

- 커밋 규칙
    - 파일 명
        -     프로젝트명
        </br>
    
    - 파일 추가
        -     [ADD] 프로젝트명
    </br>
  
    - 파일 수정
        -     [UPD] 프로젝트명
